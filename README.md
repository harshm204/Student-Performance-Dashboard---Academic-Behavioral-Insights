# 🎓 Student Performance Dashboard

A Power BI dashboard that provides insights into **student performance, attendance, and behavior** across different classes, subjects, and terms.

---

## 📌 Overview

This project helps educators and school administrators monitor:

- Academic Performance
- Attendance Trends
- Student Behavior
- Performance by Subject and Term

The dashboard is designed with interactive visuals, filters, and drill-through functionality for deeper analysis.

---

## 📂 Dataset

The report uses four datasets:

- **Students.csv** – Student details
- **Scores.csv** – Exam scores and subjects
- **Attendance.csv** – Attendance records
- **Behavior.csv** – Behavioral incidents and notes

---

## 📊 Key Metrics

- % Score
- Average Score
- Attendance %
- Behavior Count
- Performance Category (High / Medium / Low)

---

## 📄 Dashboard Pages

### 🏠 1. Main Dashboard
Provides an overall view of student performance.

**Includes:**
- Total Students
- Average Score
- Average Attendance %
- High Performers
- Behavior Incidents
- Subject-wise Performance
- Performance Trend by Term

---

### 👨‍🎓 2. Student Analysis
Detailed analysis of individual students.

**Includes:**
- Student Scores by Subject
- Attendance Summary
- Performance Category
- Student Performance Trend
- Student Detail Table

---

### 📋 3. Behavioral Analysis
Tracks and analyzes student behavior patterns.

**Includes:**
- Behavior Type Distribution
- Behavior Count by Class
- Behavior Trend by Term
- Student Behavior Records

---
